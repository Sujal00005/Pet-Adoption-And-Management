# 🐾 Pet Adoption and Management System

A full-stack Pet Adoption and Management System built using Django that allows users to browse pets, adopt pets, surrender pets, and manage adoption requests through an admin dashboard.

## ✨ Features

- User Registration & Login
- Browse Available Pets
- Pet Adoption Request
- Pet Surrender
- Admin Dashboard
- Email Notifications
- Responsive UI

## 🛠 Tech Stack

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite

## 📂 Project Structure

```
accounts/
adoptions/
dashboard/
notifications/
pet_adoption/
pets/
static/
templates/
```

## 🚀 Installation

```bash
git clone https://github.com/Sujal00005/Pet-Adoption-And-Management.git

cd Pet-Adoption-And-Management

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
